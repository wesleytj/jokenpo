TITULO DO PROJETO:Jokenpo

OBJETIVO DO PROJETO:O objetivo do desafio era treinar nosso aprendizado com Heranças em Java, porém, no meio dele eu decidi torna-lo "jogável" então desenvolvi o game para que pudesse executar.

VERSÃO ou DATA: 02/06/2020

COMO INICIAR O PROJETO: O projeto foi desenvolvido no BlueJ, por isso método mais simples é utilizando o próprio BlueJ, o método Main está na Classe Principal, basta rodar ela. Ou para executar em outra IDE, o Projeto não foi utilizando o Scanner do Java e sim a classe Teclado desenvolvida pela Professora para auxiliar no inicio do curso, então basta copiar o código da classe para que funcione corretamente a entrada de dados através do teclado.

AUTORES: Wesley Jacques

ou
Copie todo código da classe Teclado e cole em uma classe Teclado de sua IDE, para que possa utilizar a entrada de Dados solicitada pelo Teclado corretamente.
